Setup Steps [Windows Machine]
  1. Download and login to MT5
    Download MT5 from the official website and login using your broker credentials: User ID, Password, Server.
  2. Enable algorithmic trading
    Open MetaTrader 5 settings and allow Algorithmic Trading.
  3. Allow Web Request
    In MetaTrader Tools, allow Web Request for listed URLs and add https://raw.githubusercontent.com.
  4. Send your details
    Send your ID, NAME, and broker to the Dalal Street Quants Telegram channel. Wait for confirmation.
  5. Download the Algo Bot
    Download the Expert Advisor Algo Bot: dsq_v1. Double-click the bot and add it to MetaTrader.
  6. Activate the EA
    Go to Experts panel, find dsq_v1, double-click it and start the expert advisor.

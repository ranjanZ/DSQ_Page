Setup Steps Windows Machine
  1. Download MT5 & Login
    • Download MT5 from the official website: metatrader5.com
    • Login with your broker’s User ID, Password, and Server (MetaTrader 5).
    • In MetaTrader, go to Tools → Options → Expert Advisors:
      • ✅ Allow Algorithmic Trading
      • ✅ Allow Web Request for listed URL → Add https://raw.githubusercontent.com
  2. Download the Algo Bot (dsq_v1)
    • Download the EA file: dsq_v1.ex5
    • Double‑click the downloaded file → it will be added to MetaTrader.
    • In the Navigator panel (right side), find dsq_v1 under Expert Advisors.
  3. Send your details for activation

    Submit your MetaTrader ID, Name, and Broker to get Licence.

    You can also message directly on the Telegram group (preferred).
